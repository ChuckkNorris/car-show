# Car Show

To run:
 - Clone this repo `git clone https://github.com/ChuckkNorris/car-show.git`
 - Install dependencies `npm install`
 - Start the app `npm start`

To start back-end:
 - Install [.NET Core 2.x+](https://www.microsoft.com/net/learn/get-started)
 - `cd ./car-query-api`
 - `dotnet run`
